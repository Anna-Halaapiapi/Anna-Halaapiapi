### 📖🪱 My Little Bookworm
A web app helping carers track how many books their child has read - built around the goal of 1,000 books before starting school, in support of the Australian early literacy initiative. Features printable milestone certificates, user authentication via Firebase, a FastAPI backend, React frontend, and is deployed on AWS with CI/CD via GitHub Actions.

### 🐚 Simple Shell
A Unix command-line interpreter built from scratch in C. Implements process creation with `fork()`, program execution via `execve()`, PATH resolution, built-in commands, and memory-safe input parsing.

[![View Repo](https://img.shields.io/badge/Check%20It%20Out-%237F77DD?style=for-the-badge&logo=github&logoColor=white)](https://github.com/Anna-Halaapiapi/holberton_simple_shell)

### 🖨️ Custom printf
A recreation of C's `printf` function supporting `%c`, `%s`, `%d`, `%i`, and `%%` specifiers - built with variadic functions and a modular helper architecture.
